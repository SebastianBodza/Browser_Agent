# Prompt - Support Center Link Health Audit

Audit help center navigation for broken links and poor redirects.

## Objective
Check top support categories and verify that key links resolve correctly.

## Required Steps
1. Open support homepage and top categories.
2. Sample at least 40 internal help links.
3. Record final URL, status behavior, and obvious content mismatch.
4. Capture screenshots for broken/error pages.

## Output
Create `report.md` with:
- `PASS: TRUE/FALSE` (TRUE if broken-link rate is below 2%)
- broken/redirect issue table with URLs
- category-level health summary
- linked screenshots/video/logs
