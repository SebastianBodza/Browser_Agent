# Prompt - Public Sector Grant Discovery

Discover currently open grants for a given topic and region.

## Objective
Find active grants/funding calls for a target theme (example: climate AI) and region.

## Required Steps
1. Search official public-sector portals and trusted institutions.
2. Extract grant title, deadline, amount, eligibility, and official URL.
3. Exclude expired calls and duplicates.
4. Capture screenshots for each shortlisted grant.

## Output
Create `report.md` with:
- `PASS: TRUE/FALSE` (TRUE if at least 8 active grants found)
- cleaned grant table with deadlines
- shortlist of best-fit opportunities
- linked screenshots/video/logs
